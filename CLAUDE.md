# Crypto Dashboard

## Project Overview
A simple static site for viewing crypto prices. This is a front-end focused project with HTML, CSS, and JavaScript.

## Project Structure
- `index.html` - Main dashboard page for viewing crypto prices
- `login.html` - User login page
- `register.html` - User registration page
- `dashboard.html` - Dashboard view with crypto data
- `script.js` - Main JavaScript functionality
- `package.json` - Project metadata and scripts

## Getting Started
This is a static web application. To run it:
1. Open any of the HTML files directly in a browser, or
2. Use a local HTTP server: `python -m http.server 8000`

## Development Notes
- No build process required - all files are static assets
- CSS is embedded in HTML files
- JavaScript functionality is in `script.js`
- No external dependencies currently installed

## Testing
Currently no automated tests are set up. Tests can be configured with:
```
npm install --save-dev jest
npm test
```

## Browser Compatibility
This project uses standard HTML5, CSS3, and modern JavaScript. Ensure target browsers support ES6+.
